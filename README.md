# 回归线 - UWP 客户端

《明日方舟》同人文学期刊《回归线》，现在为 Windows 10 设备设计。

请注意，这是一个非官方项目，并且该项目仍然处于早期版本

## 构建指引

- 克隆本仓库
- 确保安装以下组件
    - Visual Studio 2022
    - Windows 10 SDK，版本 2004 (10.0.19041.0)
    - .NET 7 SDK
- 使用 Visual Studio 打开文件夹中的 ```AnEoT.Vintage.sln``` 文件，并运行项目

请尽情地使用吧。

## 系统要求
此应用支持的最低系统版本为 Windows 10 1703（15063）
